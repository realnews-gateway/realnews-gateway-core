# RealNews Gateway Architecture

RealNews Gateway consists of four major components:

1. **VPN Access Layer**
   - Anti-censorship network access
   - Multi-path routing
   - Automatic fallback

2. **News Aggregation Module**
   - RSS aggregation
   - Mirror sources
   - Decentralized distribution

3. **Lightweight BBS**
   - Anonymous posting
   - Moderation system
   - Lightweight storage

4. **Secure Uploader**
   - External platform upload
   - Decentralized storage (IPFS/Arweave)
   - Metadata protection
